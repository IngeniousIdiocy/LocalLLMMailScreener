Privacy Policy

Last updated: December 28, 2024

Data Access

This application requests read-only access to Gmail through the Google API:

  • Scope: gmail.readonly
  • Purpose: Read email metadata and content to determine message urgency

The application accesses email subject lines, sender information, timestamps, and message body text to evaluate whether a message requires immediate attention.

Data Handling

  • Local Processing: All email analysis is performed locally using a self-hosted LLM. No email content is sent to external AI services.
  • Minimal Storage: Only email IDs are stored to track which messages have been processed. Email content is not permanently retained.
  • No Third-Party Sharing: Email data is never shared with, sold to, or disclosed to any third parties.
  • Notifications: When urgent emails are detected, brief notification summaries are sent to the user's personal device via encrypted channels.

Data Security

  • OAuth 2.0 tokens are stored securely on the local server
  • All communication with Google APIs uses HTTPS encryption
  • The application runs on a private home network
  • No public-facing endpoints expose email data

User Rights

As the sole user of this application, you can:

  • Revoke access at any time via Google Account Permissions
  • Delete all stored data by removing the application's state files
  • Review what data is stored at any time (stored locally on your device)

Google API Services User Data Policy

This application's use and transfer of information received from Google APIs adheres to the Google API Services User Data Policy, including the Limited Use requirements.
