About This Application

EmailScreener is a personal productivity tool that monitors a Gmail inbox and uses a locally-hosted large language model (LLM) to identify important or time-sensitive messages. When an urgent email is detected, the application sends a notification to the user's mobile device.

This is a single-user, personal-use application. It is not offered as a service to the public.

How It Works

  • Email Monitoring: Periodically checks Gmail inbox for new messages
  • Local Analysis: Uses a self-hosted LLM to evaluate message urgency
  • Smart Notifications: Sends alerts only for genuinely important emails
  • Privacy-First: All processing happens locally; no data leaves your network
